*) 期间收益率按照时间加权计算：
	Y[t]=(1+Y[0])*(1+Y[1])*...*(1+Y[t-1]) - 1

*) 单个证券单个交易日Y[i]：
	Y[i] = (今日市值 - 昨日市值 + 所有利息|股利) / 昨日市值

	现金利息 = 存款利息 + 逆回购利息 - 正回购利息
	债券利息 = MAX[(今日应计利息 - 昨日应计利息),0] > 0
	股票股利 = MAX[(今日应计股利 - 昨日应计股利),0] > 0
		取MAX因为当利息支付之后会进入银行存款
		卖出价格按照平均价格估算
		
*) 活期存款利息：每季度21日结算；
*）定期存款利息：计头不计尾
*）现金头寸收益率 = 当日利息总和 / (当日定期存款本金 + 前日活期存款本金总和)
